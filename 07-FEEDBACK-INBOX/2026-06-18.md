## 02:50 — MCP-first Metabase workflow

- Status: pending
- Source: user preference (historical)
- Context: Knowledge architecture reorganization — relocated from MEMORY.md
- Feedback: Metabase operations must use MCP tools first. REST API is last resort, only for: card SQL search/extract/update, native SQL, CSV export, template-tag parameters, and datasets exceeding 200 rows. - Scope: All Metabase data operations
